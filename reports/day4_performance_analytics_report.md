# Day 4: Performance Analytics Report

## Objective
Compute risk and return metrics for mutual fund schemes and compare performance against benchmarks.

## Analysis Performed

- Daily return computation
- CAGR calculation
- Sharpe Ratio analysis
- Sortino Ratio analysis
- Alpha and Beta estimation
- Maximum Drawdown analysis
- Fund Scorecard construction
- Benchmark comparison
- Tracking Error estimation

## Key Findings

1. Daily returns exhibited near-normal distribution with occasional extreme movements.
2. Top-performing schemes achieved the highest CAGR over the analysis period.
3. Several funds generated positive alpha relative to NIFTY100.
4. Beta values identified both defensive and aggressive funds.
5. Sharpe Ratio rankings highlighted superior risk-adjusted performance.
6. Sortino Ratios provided additional downside-risk insights.
7. Maximum drawdown varied significantly across schemes.
8. Expense ratios influenced overall scorecard rankings.
9. Benchmark tracking error differed across fund categories.
10. Composite scorecard effectively ranked all 40 schemes.

- Performance_Analytics.ipynb
- alpha_beta.csv
- fund_scorecard.csv
- benchmark_comparison.png

## Conclusion

The performance analytics phase successfully quantified risk-adjusted returns, benchmark sensitivity, and overall fund quality using industry-standard investment metrics.
